Vim edits text with the keyboard only. The whole point is doing everything without a mouse, barely moving your fingers. Learn the basics before you start. During play, the HINT panel (top-right) always lists the commands for the current level.

NORMAL mode
The default state for moving & issuing commands. Letter keys act as 'commands'. (The game starts here.)
INSERT mode
Enter with i a o to actually type text. When done, always press Esc to return to NORMAL!

Move (NORMAL)

h j k lleft / down / up / right one cell — Vim's arrow keys
w b ejump to next word / previous word / end of word
0 ^ $start of line / first non-blank / end of line
f{char}leap to that char on the line ( ; repeats )
gg G 3Gfirst line / last line / line 3

Edit — "operator + motion" is Vim's grammar

xdelete one char under the cursor
d + motiondelete. dw=word, d$=to line end, dd=whole line
c + motionchange (delete then INSERT). cw=word, cc=line
y + motion / pyank (copy) / paste. yy=copy line
diw ci( ci"text objects: word / inside ( ) / inside " "

Insert & Misc

i a A ostart typing: before / after / line-end / new line below — then Esc
u · Ctrl-rundo / redo
.repeat your last change (Vim's magic!)
:help :drill :qex-commands — these never count toward your keystrokes/par
